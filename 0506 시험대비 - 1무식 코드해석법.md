## 전체적인 흐름 및 구조
1. 👤 사용자: 로그인 버튼 클릭
    
2. 📮 `Servlet`: 요청 받음 → `Service` 호출
    
3. 🧠 `Service`: 비즈니스 로직 처리 (아이디/비번 검사)
    
4. 🗃 `DAO`: DB에서 해당 사용자 정보 조회
    
5. 💾 `DB`: 실제 데이터를 DAO에게 전달
    
6. 🧠 `Service`: 로직 처리 결과 반환
    
7. 📮 `Servlet`: 결과를 `JSP`로 전달
    
8. 🖥 `JSP`: 사용자에게 화면을 보여줌
